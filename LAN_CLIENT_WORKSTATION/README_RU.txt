  Lev&Av TMS — дополнительный рабочий ПК (без сервера, только браузер)
================================================================================

На главном ПК
-------------
  Запуск только через PRODUCTION_START.bat в папке проекта LevAV_MAIN_SYSTEM.
  В окне консоли будут показаны строки вида:
    http://192.168.x.x:3000
  Запишите IP (или сфотографируйте экран).

На этом (дополнительном) ПК
----------------------------
  1) Скопируйте на Рабочий стол всю папку LAN_CLIENT_WORKSTATION
     (из проекта на главном ПК или из общей шары).

  2) Откройте в «Блокноте» файл LAN_SERVER_URL.txt
     Замените 192.168.0.1 на IP главного ПК из шага выше.
     Должна остаться одна строка, например:
       http://192.168.1.50:3000
     Сохраните файл.

  3) Запуск системы:
     • Двойной щелчок по OPEN_LEVAV_TMS.bat
       ИЛИ
     • Правый щелчок по LevAV_TMS_LAN.url → Изменить → в строке URL=
       подставьте тот же адрес → сохраните → откройте ярлык двойным щелчком.

  НЕ используйте на доп. ПК:
     localhost:3000 — это откроет «пустой» компьютер, а не сервер.
     PRODUCTION_START.bat, npm, папку проекта — на доп. ПК не нужны.
